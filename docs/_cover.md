# Pict-Nonlinearconfig

> A Pict application scaffold for multi-step, branching configuration workflows

- **Runnable shell** -- Layout, top/bottom bars, login, and routing already wired
- **Hash routing** -- Navigable views powered by pict-router as a Pict provider
- **Pict-native** -- Built on pict-application + pict-view, ready to extend
- **A starting point** -- Add your own configuration-workflow views and providers
